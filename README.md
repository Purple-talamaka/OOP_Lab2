● # ООП Лабораторная работа 2 — Объекты и Классы

  Консольное приложение на C++, демонстрирующее работу с классами, наследованием и композицией объектов.                
  ## Классы                                                                                                               - `Point` — базовый класс, точка на плоскости
  - `Circle` — наследник Point, добавляет радиус
  - `Canvas` — демонстрация композиции объектов
